Git flow

GIT clone - для того, чтобы "склонить" репозиторий
GIT pull - для того, чтобы получить изменения с репозитория
GIT chekout -b branchname - создать новою ветку и переключиться на нее
GIT add . - добавить изменения, которые сделали локально
GIT -m"commit text" - подписать их
GIT push origin brancname - запушить их на git сервер
